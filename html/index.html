Backend

In order to bring up back-end you just run 

npm install

Version control/CI

Project builds with all tests passing both on Linux and Windows. See respective badges in github repository. Builds are triggered by pull requests into [master].

Tests

If you want to run tests and see results, just go
(I have added mock for the oAuth.): 

npm test
to test eslint compliance* 
npm run _eslint
unit test coverage* 
npm run _coverage
output in JUnit style** 
npm run _junit
*) Should work both on Windows and Linux
**) works only on Linux (dev)

API & Front regression

Just run***: 

npm run _integration
***) tests assumes application is up on localhost
